== Description ==
WooPromo allow you to create beautiful promotional action banner with or without countdown to inform your customers about upcoming events or offer.


***Featured***

1. Useful to promote product, service etc.
2. 9 Premade Background Style
3. 3 Premade CountDown Style
4. Background Customize option
5. CountDown Style Customize options
6. Provide text color change option 
7. Easy to customize Easy to use
8. Shortcode Support
9. Select promo from WooCommerce Product
10. CountDown option included
11. Unlimited free update



How to use:-
very easy to use,after active plugin  just go to Dashboard -> WooPromo. then create promo shortcode and use where you like to show.

== Installation ==

**The easy way.**

Go to your WordPress Dashboard. Navigate to Plugins > Add New and then search for "WooPromo". Click on Install and then Activate the Plugin.

That's it, WooPromo Plugin is now activated on your site!

**The hard way..**

Download "WooPromo WordPress Plugin"  and then extract the zip file and just drop the contents in the wp-content/plugins/ directory of your WordPress installation and then activate the Plugin from Plugins page.
